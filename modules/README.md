# NAS Dashboard 模块文档

本目录包含NAS Dashboard各个功能模块的详细文档、脚本和实现指南。

## 模块列表

### 1. power-monitor/ - 功耗监控模块
- 功耗数据采集脚本
- 功耗监控配置
- 功耗统计和分析工具
- 相关API文档

### 2. disk-management/ - 磁盘管理模块
- 磁盘分区和格式化
- RAID管理
- LVM逻辑卷管理
- SMART监控

### 3. network-management/ - 网络管理模块
- 网络接口配置
- Wi-Fi管理
- PPPoE配置
- DNS和代理设置
- 防火墙规则

### 4. system-monitor/ - 系统监控模块
- CPU和内存监控
- 进程管理
- 服务管理
- 系统信息采集
- 温度监控
- 系统测试脚本

### 5. backup-restore/ - 备份恢复模块
- Restic备份脚本
- 定时备份配置
- 恢复操作指南
- 存储快照管理

### 6. liquidctl/ - 液冷控制模块
- 液冷设备初始化
- RGB灯光控制
- 温度和风扇控制
- 状态监控

### 7. storage-pool/ - 存储池管理模块
- MergerFS多磁盘合并
- 存储策略配置
- 磁盘动态管理
- 存储池监控

### 8. application-system/ - 应用系统模块
- 应用包管理
- 应用中心功能
- 应用安装配置
- 应用模板工具

### 9. user-management/ - 用户管理模块
- 用户账户管理
- 用户组管理
- SSH密钥管理
- 配额管理
- SMB用户管理

### 10. permission-management/ - 权限管理模块
- 文件权限管理
- ACL权限控制
- 共享权限设置
- 权限继承机制

### 11. share-management/ - 共享管理模块
- SMB/CIFS共享
- NFS共享
- WebDAV支持
- 共享权限管理
- 回收站功能

### 12. plugin-system/ - 插件系统模块
- 前端插件架构
- 后端插件API
- 插件开发SDK
- 应用包管理

## 使用说明

每个模块都包含：
- **README.md** - 模块概述和使用说明
- **scripts/** - 相关脚本文件
- **docs/** - 详细文档
- **config/** - 配置示例

## 贡献指南

添加新模块时，请遵循以下结构：
1. 在modules/下创建新的模块目录
2. 添加README.md说明模块功能
3. 按功能分类组织脚本和文档
4. 在本文件中添加模块说明
