# NAS Dashboard 新功能使用指南

## 🎉 欢迎使用升级版NAS Dashboard

您的NAS Dashboard已经完成了重要的系统升级，新增了**控制面板**和**通知系统**两大核心功能，对标群晖DSM的使用体验。

## 🚀 快速开始

### 1. 访问控制面板

#### 方法一：通过Dock栏
1. 查看桌面底部的Dock栏
2. 找到**齿轮图标**（控制面板）
3. 点击图标打开控制面板

#### 方法二：通过桌面图标
1. 桌面中央的快速访问图标
2. 点击任意管理图标

#### 控制面板功能说明

**🔧 通用设置**
- **系统名称**: 设置NAS的主机名
- **时区设置**: 选择您所在的时区
- **系统语言**: 切换界面语言
- **自动更新**: 启用系统自动更新
- **维护时间**: 设置系统维护时间窗口

**🌐 网络设置**
- **主机名**: 设置网络主机名
- **DNS配置**: 配置域名服务器
- **代理设置**: 设置网络代理
- **端口配置**: 网络服务端口设置

**🛡️ 安全设置**
- **防火墙**: 启用/禁用系统防火墙
- **SSL/TLS**: 启用HTTPS加密连接
- **防暴力破解**: 自动封禁恶意IP
- **会话超时**: 设置用户会话超时时间

**💾 存储设置**
- **自动挂载**: 系统启动时自动挂载磁盘
- **省电模式**: 硬盘空闲时降速省电
- **S.M.A.R.T.监控**: 硬盘健康状态监控

**🔔 通知设置**
- **邮件通知**: 发送系统通知到邮箱
- **系统通知**: 启用桌面通知
- **通知声音**: 通知提示音

**🎨 外观设置**
- **界面主题**: 选择不同的界面主题
- **桌面壁纸**: 启用/禁用桌面背景
- **界面动画**: 启用/禁用过渡动画
- **界面密度**: 调整界面元素紧凑程度

### 2. 使用通知系统

#### 查看通知
1. 桌面右上角的**铃铛图标**
2. 显示当前未读通知数量
3. 点击图标打开通知中心

#### 通知类型说明

**📋 通知分类**
- **系统通知**: 系统相关消息
- **存储通知**: 磁盘和存储消息
- **网络通知**: 网络连接消息
- **安全通知**: 安全相关警告
- **用户通知**: 用户活动消息
- **应用通知**: 应用程序消息
- **文件通知**: 文件操作消息

**🎨 通知类型**
- **ℹ️ 信息**: 一般信息通知
- **⚠️ 警告**: 警告性通知
- **❌ 错误**: 错误通知
- **✅ 成功**: 成功操作通知

#### 通知操作
- **标记已读**: 点击信封图标切换已读/未读状态
- **删除通知**: 点击垃圾桶图标删除通知
- **执行操作**: 某些通知包含操作按钮，可直接执行

### 3. 配置管理

#### 导出配置
1. 打开控制面板
2. 点击右上角的**下载图标**
3. 选择是否包含元数据
4. 点击**导出**按钮
5. 配置文件将自动下载

#### 导入配置
1. 打开控制面板
2. 点击右上角的**上传图标**
3. 粘贴配置JSON内容
4. 点击**导入**按钮
5. 确认导入操作

**⚠️ 注意**: 导入配置将覆盖当前设置，请谨慎操作。

## 💡 使用技巧

### 控制面板技巧

#### 快速查找设置
1. 使用顶部的**搜索框**
2. 输入设置关键词（如"网络"、"安全"）
3. 系统将显示相关设置项

#### 高级设置
1. 某些分类包含**高级设置**部分
2. 默认隐藏，点击**显示**按钮展开
3. 包含专家级设置选项

#### 重置功能
- **重置分类**: 重置当前分类的所有设置
- **重置全部**: 重置所有设置为默认值

### 通知系统技巧

#### 通知过滤
1. 打开通知中心
2. 使用顶部的**过滤器**
3. 按类型或状态过滤通知

#### 通知搜索
1. 在通知中心使用**搜索框**
2. 输入关键词搜索通知内容
3. 支持标题和消息搜索

#### 通知设置
1. 点击通知中心的**设置图标**
2. 配置通知行为：
   - 启用/禁用通知
   - 调整显示位置
   - 设置显示时长
   - 配置分类通知

## 🔧 系统要求

### 浏览器兼容性
- **推荐**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **最低**: Chrome 80+, Firefox 80+, Safari 12+

### 系统要求
- **内存**: 最低2GB RAM
- **存储**: 最低10GB可用空间
- **网络**: 建议千兆网络

## 📞 获取帮助

### 问题排查

#### 控制面板无法打开
1. 刷新浏览器页面
2. 清除浏览器缓存
3. 检查浏览器控制台错误

#### 通知不显示
1. 检查通知设置是否启用
2. 确认浏览器通知权限
3. 查看特定分类是否启用

#### 设置保存失败
1. 检查网络连接
2. 验证设置值是否有效
3. 查看浏览器控制台错误信息

### 技术支持
- **文档**: 查看系统文档
- **社区**: 访问用户论坛
- **问题反馈**: 提交问题报告

## 🎓 进阶使用

### 键盘快捷键
- **Ctrl/Cmd + ,**: 打开控制面板
- **Ctrl/Cmd + N**: 打开通知中心
- **Ctrl/Cmd + F**: 聚焦搜索框

### 自定义主题
1. 打开控制面板
2. 进入**外观设置**
3. 选择或创建自定义主题
4. 应用主题更改

### 配置备份策略
- **定期备份**: 每月导出一次配置
- **版本管理**: 保存不同版本的配置文件
- **安全存储**: 将配置文件存储在安全位置
