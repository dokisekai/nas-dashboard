## 🔧 技术实现细节

### 前端技术栈
- Vue 3 + TypeScript
- Pinia (状态管理)
- Vue Router (路由)
- Tailwind CSS (样式)
- ApexCharts (图表)
- Vue Draggable (拖拽)

### 后端技术栈
- Go + Gin框架
- JWT认证
- WebSocket (实时通信)
- 系统监控库 (gopsutil)
- Docker管理
