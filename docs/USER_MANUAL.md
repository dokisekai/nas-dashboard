## 存储池管理

### 创建存储池

1. 进入 **存储管理** → **存储池**
2. 点击 **创建存储池** 按钮
3. 选择存储池类型:
   - **MergerFS**: 合并多个磁盘空间
   - **Btrfs**: 支持快照和压缩
   - **LVM**: 逻辑卷管理
4. 添加要包含的磁盘
5. 配置 MergerFS 策略:
   - **min space most**: 优先使用空间最多的磁盘
   - **most free space**: 优先使用空闲空间最多的磁盘
   - **mspmfs**: 自适应多策略
6. 设置挂载点和描述
7. 确认创建

### 存储池操作

#### 查看存储池状态
- 总容量和可用空间
- 磁盘列表和健康状态
- 读写使用情况

#### 管理存储池磁盘
- **添加磁盘**: 动态扩展存储池容量
- **移除磁盘**: 从存储池移除磁盘
- **平衡数据**: 重新分布数据到各磁盘

#### 挂载/卸载存储池
- **挂载**: 使存储池可用
- **卸载**: 安全地停止存储池访问

## 磁盘管理

### 物理磁盘

#### 磁盘信息
- **设备名称**: /dev/sda, /dev/sdb 等
- **容量**: 磁盘总大小
- **型号**: 磁盘品牌和型号
- **健康状态**: S.M.A.R.T. 健康度
- **温度**: 当前磁盘温度

#### 磁盘操作
- **挂载/卸载**: 访问磁盘数据
- **格式化**: 创建文件系统
- **测试**: 运行磁盘性能测试

### 分区管理

#### 分区编辑器
- **查看分区表**: 显示现有分区
- **创建分区**: 在未分配空间创建新分区
- **删除分区**: 删除不需要的分区
- **调整大小**: 修改分区大小

#### 分区类型
- **主分区**: 标准主分区
- **逻辑分区**: 扩展分区中的逻辑分区
- **扩展分区**: 包含多个逻辑分区

#### 文件系统
- **ext4**: Linux 推荐文件系统
- **xfs**: 高性能文件系统
- **btrfs**: 支持快照和压缩
- **NTFS**: Windows 兼容
- **FAT32**: 最大兼容性

### RAID 管理

#### 创建 RAID
1. 进入 **磁盘管理** → **RAID**
2. 点击 **创建 RAID**
3. 选择 RAID 级别:
   - **RAID 0**: 条带化，提升性能
   - **RAID 1**: 镜像，数据冗余
   - **RAID 5**: 分布式奇偶校验
   - **RAID 6**: 双奇偶校验
   - **RAID 10**: 镜像和条带化
4. 选择要包含的磁盘
5. 配置 RAID 参数
6. 创建并等待同步完成

#### RAID 维护
- **查看状态**: RAID 阵列状态
- **添加磁盘**: 扩展 RAID 容量或替换故障盘
- **移除磁盘**: 从 RAID 移除磁盘
- **重建阵列**: 恢复 RAID 冗余

### LVM 管理

#### 物理卷 (PV)
- **创建物理卷**: 初始化磁盘为物理卷
- **查看物理卷**: 列出所有物理卷
- **移除物理卷**: 从 LVM 移除

#### 卷组 (VG)
- **创建卷组**: 组合多个物理卷
- **扩展卷组**: 添加更多物理卷
- **查看卷组**: 卷组容量和状态

#### 逻辑卷 (LV)
- **创建逻辑卷**: 从卷组分配空间
- **调整大小**: 扩展或缩小逻辑卷
- **快照**: 创建逻辑卷快照

## 配额管理

### 用户配额

#### 设置用户配额
1. 进入 **配额管理** → **用户配额**
2. 选择要设置配额的用户
3. 配置配额参数:
   - **软限制**: 警告阈值
   - **硬限制**: 最大使用限制
   - **宽限期**: 超过软限制的宽限时间
   - **路径**: 应用配额的目录
4. 保存配置

#### 查看用户配额
- **使用情况**: 当前使用空间
- **使用百分比**: 相对于限制的百分比
- **状态**: 正常、警告或超限

#### 配额操作
- **修改配额**: 调整配额限制
- **删除配额**: 移除用户配额
- **查看详情**: 用户配额详细信息

### 组配额

#### 管理组配额
- **创建组配额**: 为组设置空间限制
- **修改组配额**: 调整组配额参数
- **删除组配额**: 移除组配额

#### 组成员管理
- **添加成员**: 向组添加用户
- **移除成员**: 从组移除用户
- **继承设置**: 组配额继承选项

### 配额报告

#### 使用报告
- **总体统计**: 配额使用总览
- **用户排行**: 按使用量排序
- **告警用户**: 超限或接近限制的用户

#### 趋势分析
- **使用趋势**: 历史使用情况
- **增长预测**: 预计何时达到限制
- **建议操作**: 基于分析的建议

### 配额告警

#### 告警配置
- **告警类型**: 用户配额、组配额、磁盘空间
- **触发条件**: 超过阈值或条件
- **严重程度**: 信息、警告、严重
- **通知方式**: 邮件、系统通知、Webhook

#### 告警历史
- **告警记录**: 历史告警信息
- **处理状态**: 告警处理状态
- **告警统计**: 告警频率统计

## 常见问题

### 登录问题

#### 忘记密码
1. 使用管理员账户登录
2. 进入用户管理
3. 选择忘记密码的用户
4. 点击重置密码
5. 设置新密码

#### 账户被锁定
- 等待锁定时间过期
- 或联系管理员解锁

### 存储问题

#### 存储池无法挂载
1. 检查磁盘状态
2. 查看系统日志
3. 尝试重新挂载
4. 必要时重启系统

#### MergerFS 错误
1. 检查 MergerFS 是否安装
2. 验证磁盘路径
3. 检查磁盘格式
4. 查看错误日志

### 性能问题

#### 系统缓慢
1. 查看资源使用情况
2. 检查进程和服务
3. 查看磁盘 I/O
4. 考虑升级硬件

#### 存储速度慢
1. 检查磁盘健康
2. 验证连接速度
3. 优化存储池配置
4. 考虑使用 SSD 缓存

### 网络问题

#### 无法访问网络共享
1. 检查网络连接
2. 验证 SMB 服务状态
3. 检查防火墙设置
4. 验证用户权限

#### 网络速度慢
1. 检查网络接口速度
2. 验证网线和交换机
3. 检查网络负载
4. 优化网络设置

### 数据安全

#### 数据丢失
1. 检查备份
2. 使用数据恢复工具
3. 联系专业数据恢复服务

#### RAID 故障
1. 检查 RAID 状态
2. 更换故障磁盘
3. 重建 RAID 阵列
4. 验证数据完整性
