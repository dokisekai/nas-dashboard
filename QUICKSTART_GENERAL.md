# NAS Dashboard 快速开始指南

欢迎使用 NAS Dashboard！本指南将帮助您快速上手系统的主要功能。

## 5分钟快速配置

### 第一步：登录系统

1. 打开浏览器，访问 NAS Dashboard 地址
2. 输入默认管理员账户：
   - 用户名：`admin`
   - 密码：`admin`（首次登录后请立即修改）

### 第二步：创建存储池

1. 点击左侧菜单的 **存储管理**
2. 选择 **存储池** 选项卡
3. 点击 **创建存储池** 按钮
4. 按照向导步骤操作：
   - 输入存储池名称（如 "main-pool"）
   - 选择存储池类型（推荐 MergerFS）
   - 选择要包含的磁盘
   - 选择挂载策略（推荐 "min space most"）
   - 设置挂载点（如 "/mnt/main-pool"）
5. 点击 **创建** 完成存储池创建

### 第三步：创建共享文件夹

1. 在存储池页面点击 **创建共享文件夹**
2. 输入文件夹名称（如 "documents"）
3. 选择访问权限：
   - 公开：所有用户可访问
   - 私有：仅授权用户可访问
4. 设置用户权限
5. 点击 **创建** 完成共享文件夹创建

### 第四步：添加用户

1. 点击左侧菜单的 **用户管理**
2. 选择 **用户** 选项卡
3. 点击 **创建用户** 按钮
4. 填写用户信息：
   - 用户名（如 "john"）
   - 邮箱地址
   - 设置密码
   - 选择用户组
5. 点击 **保存** 完成用户创建

### 第五步：设置用户配额

1. 在用户管理页面，选择刚创建的用户
2. 点击 **设置配额** 按钮
3. 配置配额参数：
   - 选择要应用配额的路径
   - 设置软限制（如 10GB）
   - 设置硬限制（如 15GB）
   - 设置宽限期（如 7 天）
4. 点击 **保存** 完成配额设置

## 核心功能简介

### 🗄️ 存储管理

**存储池**：将多个物理磁盘合并为一个大容量存储空间
- 支持 MergerFS、Btrfs、LVM 等多种存储类型
- 动态添加/移除磁盘
- 自动负载均衡

**共享文件夹**：在存储池中创建共享目录
- 细粒度权限控制
- 支持 SMB、NFS、AFP 等协议
- 快照和版本控制

### 📊 系统监控

**实时监控**：系统资源使用情况实时显示
- CPU、内存、磁盘使用率
- 网络流量监控
- 温度监控

**进程管理**：管理系统运行进程
- 查看所有运行进程
- 结束卡死的进程
- 调整进程优先级

**服务管理**：控制系统服务
- 启动/停止系统服务
- 设置开机自启动
- 查看服务状态和日志

### 💾 磁盘管理

**物理磁盘**：管理所有物理存储设备
- 查看 S.M.A.R.T. 健康状态
- 磁盘性能测试
- 温度和错误率监控

**分区管理**：图形化分区编辑
- 创建/删除/调整分区
- 支持多种文件系统
- 分区信息查看

**RAID 管理**：创建和管理 RAID 阵列
- 支持 RAID 0/1/5/6/10
- 磁盘添加和移除
- 阵列重建和监控

### 👥 用户和权限

**用户管理**：管理系统用户账户
- 创建/删除用户账户
- 用户权限设置
- 用户组管理

**配额管理**：设置用户和组的空间配额
- 磁盘空间配额
- 文件数量配额
- 使用情况和告警

## 常见操作

### 如何访问共享文件夹？

**Windows 用户**：
1. 打开文件资源管理器
2. 在地址栏输入：`\\your-nas-ip\share-name`
3. 输入用户名和密码
4. 访问共享文件

**Mac 用户**：
1. 打开 Finder
2. 按 `Cmd + K` 连接服务器
3. 输入：`smb://your-nas-ip/share-name`
4. 输入用户名和密码
5. 挂载共享

**Linux 用户**：
```bash
# 挂载 SMB 共享
sudo mount -t cifs //your-nas-ip/share-name /mnt/share -o username=user,password=pass
```

### 如何监控系统性能？

1. 点击左侧菜单的 **系统监控**
2. **概览** 选项卡显示：
   - CPU 使用率和历史趋势
   - 内存使用情况和进程列表
   - 磁盘 I/O 和网络流量
3. **温度** 选项卡查看硬件温度
4. **告警** 选项卡配置性能告警

### 如何备份重要数据？

1. **创建快照**：
   - 进入存储池管理
   - 选择共享文件夹
   - 点击创建快照

2. **配置自动备份**：
   - 进入 **系统设置** → **备份和恢复**
   - 创建备份计划
   - 选择备份源和目标
   - 设置备份频率

3. **远程备份**：
   - 配置远程存储服务
   - 设置自动同步

### 如何排查系统问题？

1. **查看系统日志**：
   - 进入 **系统设置** → **系统日志**
   - 选择日志类型
   - 搜索错误信息

2. **检查服务状态**：
   - 进入 **系统监控** → **服务管理**
   - 查看服务运行状态
   - 重启失败的服务

3. **运行诊断工具**：
   - 进入 **系统设置** → **诊断**
   - 运行系统诊断
   - 查看诊断报告
