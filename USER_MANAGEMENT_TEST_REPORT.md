## 🔍 问题分析

### 主要问题：用户信息映射不一致

**问题描述**：
- JWT Token中的用户名是数据库用户（admin）
- API实现试图用此用户名查找系统用户（user.Lookup）
- admin用户不在/etc/passwd中，导致查找失败

**影响范围**：
- `/api/users/me` 端点无法使用
- 依赖当前用户信息的功能受限

**建议修复**：
```go
// 应该从数据库获取用户信息，而不是系统用户
func GetCurrentUser(c *gin.Context) {
    userID, exists := c.Get("userID")
    if !exists {
        c.JSON(401, gin.H{"error": "Not authenticated"})
        return
    }

    // 从数据库查询用户
    var user models.User
    if err := db.First(&user, userID).Error; err != nil {
        c.JSON(404, gin.H{"error": "User not found"})
        return
    }

    c.JSON(200, user)
}
```
