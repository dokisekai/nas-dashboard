# Docker 管理应用功能文档

## 功能概述

Docker管理应用已成功添加到NAS控制面板，提供完整的Docker容器、镜像、网络和卷管理功能。

## 新增功能

### 1. 桌面集成
- 在桌面底部Dock栏添加了"Docker管理"图标
- 点击图标即可启动Docker管理应用
- 路由路径: `/docker`

### 2. 容器管理
#### 基础功能
- 查看所有容器列表（运行中和已停止）
- 实时显示容器状态和资源使用情况
- 启动、停止、重启容器
- 删除容器
- 查看容器日志
- 容器详情查看

#### 高级功能
- **批量操作**: 支持多选容器进行批量启动、停止、重启
- **实时监控**: 显示CPU和内存使用情况
- **日志查看**: 在模态框中查看容器日志
- **终端接入**: 为容器提供终端访问功能（开发中）
- **容器详情**: 查看完整的容器配置信息
  - 基本信息（ID、镜像、状态等）
  - 网络配置
  - 端口映射
  - 卷挂载信息

#### 过滤功能
- 全部容器
- 仅运行中的容器
- 仅已停止的容器

### 3. 镜像管理
#### 功能特性
- 查看本地所有镜像
- 拉取新镜像
- 删除镜像
- 从镜像运行容器
- 显示镜像大小和创建时间

#### 操作说明
**拉取镜像:**
1. 点击"拉取镜像"按钮
2. 输入镜像名称（如 `nginx:alpine`）
3. 确认拉取

**运行容器:**
- 点击镜像的"运行"按钮
- 输入容器名称
- 系统会创建并启动新容器

### 4. 网络管理
#### 功能特性
- 查看所有Docker网络
- 创建自定义网络
- 删除网络（除了系统预设网络）
- 查看网络详情

#### 网络类型
- **bridge**: 桥接网络，默认类型
- **overlay**: 覆盖网络，用于Swarm
- **macvlan**: MAC地址虚拟化网络

#### 创建网络
1. 点击"创建网络"按钮
2. 填写网络名称
3. 选择驱动类型
4. 可选：配置子网和网关（仅bridge网络）
5. 确认创建

### 5. 卷管理
#### 功能特性
- 查看所有Docker卷
- 创建新卷
- 删除未使用的卷
- 查看卷详情
- 显示卷使用情况

#### 卷状态
- **使用中**: 卷正在被容器使用
- **未使用**: 卷未被使用，可以安全删除

## 使用说明

### 容器管理示例

**启动已停止的容器:**
1. 在容器列表中找到已停止的容器
2. 点击绿色的"启动"按钮
3. 等待容器启动完成

**查看容器日志:**
1. 点击容器的"日志"图标
2. 在弹出的模态框中查看实时日志
3. 可以选择复制日志内容

**批量重启多个容器:**
1. 勾选要重启的容器（使用复选框）
2. 点击"批量重启"按钮
3. 确认操作

### 镜像管理示例

**拉取最新版nginx:**
```
镜像名称: nginx:latest
```

**拉取特定版本的PostgreSQL:**
```
镜像名称: postgres:15-alpine
```

### 网络管理示例

**创建自定义bridge网络:**
```
网络名称: mynetwork
驱动: bridge
子网: 172.20.0.0/16
网关: 172.20.0.1
```

### 卷管理示例

**创建持久化存储卷:**
```
卷名称: app-data
驱动: local
```

## API接口

所有Docker管理功能通过以下API端点实现：

```
GET    /api/docker/containers              - 获取容器列表
POST   /api/docker/containers/:id/start   - 启动容器
POST   /api/docker/containers/:id/stop    - 停止容器
POST   /api/docker/containers/:id/restart - 重启容器
DELETE /api/docker/containers/:id         - 删除容器
GET    /api/docker/containers/:id/logs   - 获取容器日志
POST   /api/docker/containers/:id/exec   - 在容器中执行命令
GET    /api/docker/images                 - 获取镜像列表
DELETE /api/docker/images/:id            - 删除镜像
POST   /api/docker/images/pull            - 拉取镜像
GET    /api/docker/networks               - 获取网络列表
POST   /api/docker/networks               - 创建网络
DELETE /api/docker/networks/:id           - 删除网络
GET    /api/docker/volumes                - 获取卷列表
POST   /api/docker/volumes                - 创建卷
DELETE /api/docker/volumes/:name         - 删除卷
```

使用Docker管理功能需要：
- 登录NAS控制面板
- 具有管理员权限
- Docker服务正常运行

## 注意事项

1. **系统网络保护**: `bridge`、`host`、`none`网络不能删除
2. **使用中的卷**: 正在被容器使用的卷无法删除
3. **运行中的容器**: 删除容器前需要先停止
4. **镜像依赖**: 删除镜像前确保没有容器在使用
5. **网络连接**: 删除网络前确保没有容器连接到该网络
